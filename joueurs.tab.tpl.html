Critères de classement

Afin de guider votre recherche sur les joueurs présents, des critères sont mis à votre disposition. Il est en effet possible de classer les joueurs en fonction de :

  • L'odre alphabétique
  • La note moyenne
  • Le nombre de likes
  • Le nombre de bagues obtenues
  • Le nombre d'années passées en NBA
  • L'année de la Draft

Liste des joueurs

Ci-dessous se trouve le tableau des joueurs ainsi que leurs caractéristiques respectives. A partir de ce tableau, plusieurs actions sont possibles :

  • Commenter : Pour poster ou regarder les commentaires sur le joueur de votre choix, veuillez cliquer sur le nom du joueur concerné. Vous serez alors redirigé sur la page comprenant les commentaires
  • Liker un joueur : cliquer sur le pouce correspondant. Si celui-ci est noir, cela signifie que vous avez déjà liké. Vous pouvez annulé votre choix en re-cliquant dessus. Attention ! Vous devez être connécté pour liker, sinon vous serez redirigé sur la page d'inscription.

Un formulaire est mis à votre disposition pour classer les joueurs en fonction du critère de votre choix.
